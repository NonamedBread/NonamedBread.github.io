세 변의 길이를 가지고 삼각형이 될 수 있는지 판별하는 알고리즘 
제일 긴 한변의 길이가 나머지 두 변의 길이의 합보다 작거나 같아야 한다.

```javascript
function solution(a, b, c){
                let answer = 'YES', max;
                let sum = a + b + c;
                if(a>b) max=a;
                else max = b;
                if(c>max) max = c;
                if ((sum-max)<= max) answer = "NO";
                return answer;
            }
            console.log(solution(6, 7, 11));
            console.log(solution(13, 33, 17));
```
